# Analysis of Netflix Userbase - Exploratory Data Analysis
****

## About this project

From a database found in <a href="https://www.kaggle.com/datasets/arnavsmayan/netflix-userbase-dataset" target="_blank">Kaggle</a>, this analysis is about the behaviour from multiple users across the world. The range of date is between 2021 to July/2023.

Some of features that we can found on the database:

| Column | Information |
|--------|-------------|
| **User ID** | Unique ID of users |
| **Subscription Type** | Three types of subscription: Basic, Standard and Premium |
| **Monthly Revenue** | Expected revenue from each User |
| **Join Date** | Date where here sign up Netflix |
| **Last Payment Date** | Last time the user paid the subscription |
| **Country** | Where the user comes from |
| **Age** | How old is the user |
| **Gender** | Male or Female |
| **Device** | Where user watch Netflix |
| **Plan Duration** | This database only considered 1 month plan |

## Analysis

Using python with plot libraries (Matplotlib and Seaborn), the analysis is exploratory, in order to understand the behavior of these users and how it impacts the revenue over time.
